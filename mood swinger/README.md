# Face Detection Program

This is a mood detection program that uses your webcam to detect faces and analyze emotions in real-time using OpenCV and FER (Facial Expression Recognition).

## Requirements
- Python 3.x
- OpenCV (opencv-python)
- NumPy
- FER (fer)
- TensorFlow

## How to Run
1. Make sure your webcam is connected and working
2. Run the script:
   ```
   python face_detection.py
   ```
3. The program will open your webcam and start detecting faces
4. Press 'q' to quit the program

## Features
- Real-time face detection using webcam
- Uses OpenCV's Haar Cascade Classifier
- Draws green rectangles around detected faces
